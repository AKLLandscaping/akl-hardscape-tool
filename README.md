# AKL Hardscape Master Tool

This is a custom Streamlit web application for quoting hardscape jobs, including:
- Walkways
- Retaining Walls
- Steps
- Fire Pits

Each section calculates material, labor, equipment, and travel costs with Shaw Brick pricing.
